# Balatro攻略指南网站

这是一个专业的《小丑牌》(Balatro)游戏攻略网站，为全球玩家提供从零基础到高手进阶的完整教学内容。

## 网站结构

```
/home/engine/project/
├── index.html          # 主页面 - 完整攻略内容
├── calculator.html     # 计算器页面 - 收益计算工具
├── combos.html         # 组合指南页面 - 卡牌搭配策略
├── styles.css          # 主样式文件
├── calculator.css      # 计算器页面样式
├── combos.css          # 组合指南页面样式
├── script.js           # 主交互脚本
├── calculator.js       # 计算器功能脚本
└── combos.js           # 组合指南交互脚本
```

## 主要功能

### 🏠 主页面 (index.html)
- **完整游戏教学**：从基础入门到高级技巧
- **系统化学习路径**：0基础 → 了解规则 → 熟悉玩法 → 进阶技巧 → 大神级别
- **图文并茂内容**：丰富的可视化展示和互动元素
- **响应式设计**：完美适配各种设备

### 🧮 计算器页面 (calculator.html)
- **筹码计算器**：精确计算手牌基础筹码价值
- **倍数计算器**：计算各类倍数加成
- **总收益计算**：综合计算最终收益
- **盲注对比分析**：评估通过概率
- **预设组合**：常用组合快速加载
- **最优方案推荐**：基于目标盲注的策略建议

### 🃏 组合指南页面 (combos.html)
- **热门经典组合**：经过验证的最强搭配
- **分类筛选**：新手、进阶、专家、专项流派
- **详细分析**：每个组合的操作步骤和适用场景
- **组合构建器**：自定义卡牌搭配工具
- **评分系统**：从协调性、爆发力、稳定性、成长性四个维度评估
